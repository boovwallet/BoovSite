# Exporting The Portfolio PDF

1. Run the site with `npm run dev`.
2. Open `http://localhost:3000` in Chrome.
3. Choose File -> Print.
4. Destination: Save as PDF.
5. Paper size: A4.
6. Enable background graphics.
7. Save the PDF for Canvas submission.

The print stylesheet hides navigation, video, and embedded PDF frames while keeping the rubric sections paginated.
